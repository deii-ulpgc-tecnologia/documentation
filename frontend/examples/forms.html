Formularios

input type="text" implica cualquier caracter, se utiliza para campos que van a ser pequeños.



input type="number" implica caracter numéricos.



input type="email" se utiliza para especificar correos electrónicos.



input type="phone" se utiliza para especificar correos electrónicos.



input type="pass" se utiliza para ocultar el texto que se introduce.



textarea se utiliza para escribir un gran número de palabras. Normalmente para comentarios, opiniones, etc.

cols: número de columnas, rows: número de filas




input type="pass" se utiliza para ocultar el texto que se introduce.



input type="radio" genera una radio-button. Esta puede ser pulsada clicando sobre el propio cuadrado, o sobre la etiqueta asignada para mayor UX. Se usa para dar al usuario la posibilidad de elegir 1 entre varias opciones





input type="checkbox" genera una checkbox. Se usa para dar al usuario la posibilidad de elegir entre 0 y N opciones.





input type="color" se utiliza para seleccionar un color.



input type="date" se utiliza para seleccionar una fecha.



input type="file" se utiliza para seleccionar un fichero del pc.
